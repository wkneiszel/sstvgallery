A Gallery of User-Submitted Images, Uploaded by Radio

This is my SSTV Gallery. It is a system consisting of a web server and a computer-controlled VHF radio station which can receive images transmitted over the air using the Robot36 SSTV mode. Anyone can upload an image to the site by finding the receiver and transmitting a properly encoded image at the frequency 145.500 MHz. Below is a diagram of how the system functions.

System Diagram

DTMF Command Reference:

DTMF Command Station Response
1Listen for SSTV
2Transmit the current view from the station's onboard camera
3Transmit the most recent image from the database
4Transmit a random image from the database

Made using:

  • Django: The web framework for perfectionists with deadlines
  • Django REST framework used for the site's browsable api
  • Robot36, a program written in C for encoding and decoding SSTV Images
  • FFmpeg
  • dtmf-stream written by Logan Pulley, a Python module for decoding DTMF tones

Additional Info

I will be updating the site with the receiver's location. I may in the future have multiple receivers at various locations if possible.

Special thanks to Logan Pulley for writing dtmf-stream, the DTMF decoding module used to write the station code, as well as hosting the site on his server.
